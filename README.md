
##Docker lnmp 环境

#####一个使用docker构建的PHP环境

* Nginx/1.9.2
* PHP v5.5.9
* php-fpm
* mysql-server/5.6 

##使用方法

```php
$ git clone https://github.com/dolia/docker-lnmp
$ tar zxvf master.zip
$ cd master
$ docker-composer up
``` 
docker和docker-compose 安装文档： https://doc.docker.io
